This is my resume for Linux Character Device Driver 

Environment:Ubuntu with kernel version 2.6 or later than

Opperation Flow: 
	su root (need root password)
	make
	./dev_load
	echo "(your name)" > /dev/resume
	dmesg | tail -5
	cat /dev/resume
	dmesg | tail -5
	make clean
	./dev_unload
